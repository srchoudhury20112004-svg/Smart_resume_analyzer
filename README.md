## Quick start (backend)
1. cd backend
2. python -m venv venv
3. source venv/bin/activate  # or venv\Scripts\activate
4. pip install -r requirements.txt
5. python app.py

## Quick start (frontend)
1. cd frontend
2. npm install
3. npm start

## Colab
Open `notebooks/Smart_Resume_Analyzer.ipynb` in Google Colab to run demo notebooks.
